### Setup & Dependencies

- rust toolchain for WASM (cargo, rustup, wasm32-unknown-unknown, wasm-pack)
- emscripten
- nodejs
- wrangler
- wasmer

Please run setup.sh on a Ubuntu machine (VM/Container/Host) for quickly setting up the environment
